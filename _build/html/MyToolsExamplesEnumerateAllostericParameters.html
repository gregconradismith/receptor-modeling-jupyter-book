Examples of using MyTools.ipynb - allosteric parameters in receptor dimers#

The file MyTools.ipynb is includes the The function enumerate_allosteric_parameters(G) takes a graph G and returns the $\kappa$ and $\eta$ values for a dimer model with the topology of the reduced graph power $G^{(2)}$. The default method uses $e_1, e_2, \ldots , e_v$. Using method='alpha' gives single digit edge labels ($b,c,\ldots$).

%run MyTools.ipynb

Enumerating allosteric parameters for a 5-state receptor model with the topology of the house graph.#
