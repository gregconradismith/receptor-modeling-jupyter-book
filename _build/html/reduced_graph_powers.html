Reduced Graph Powers#

For a receptor model given by state-transition diagram \(G\), the state-transition diagram for the dimer is the reduced graph power \(G^{(2)}\). This expanded state-transition diagram for the receptor dimer can be created in Sagemath by constructing the Cartesian product \(G \Box G\) by merging vertices that are equivalent up to symmetry.

G = graphs.PathGraph(4)
G.show(figsize=2)

GG = G.cartesian_product(G)
pos = {v: v for v in GG.vertices(sort=True)}
GG.set_pos(pos)
GG.show(figsize=4,vertex_size=1000)

The following code block merges vertices of the Cartesian product graph \(G \Box G\) (GG) to produce the reduced graph power \(G^{(2)}\) (G2).

G2 = GG.copy()
for v in G2.vertices(sort=True):
    for u in G2.vertices(sort=True):
        sv=tuple(sorted(v))
        su=tuple(sorted(u))
        if v!=u and sv==su and v<u and G2.has_vertex(v) and G2.has_vertex(u):
            G2.merge_vertices([v,u])
G2.show(figsize=5,vertex_size=1000)

def Cartesian_power(G=Graph(), k=2):
    # Make Cartesian power G^k (unreduced)
    Gk=G.copy()
    for i in range(k-1):
        Gk = Gk.cartesian_product(G)
    # Make each vertex a tuple
    vflat=list(range(Gk.order()));
    for i in range(Gk.order()):
        v=Gk.vertices(sort=True)[i]
        vflat[i]=tuple(flatten(v))
    Gk.relabel(vflat)
    return Gk
